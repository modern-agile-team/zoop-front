### ✅ 주요 목적

사용자가 실시간으로 문제를 보고 정답을 입력하여 점수를 획득하도록 한다.

---

### 🧩 기능 명세

### 🔹 1. 문제 출제

- 문제는 게임 진행 중 랜덤으로 출제됨
- 한 문제당 제한 시간은 N초 (남은 시간 실시간 표시)

### 🔹 2. 정답 제출

- 입력창에 정답 입력 후 엔터 또는 제출 버튼 클릭 시 제출
- 빈 문자열은 제출 불가능
- 정답을 틀리면 감점 (몇 점 감점인지 룰 명시 필요)
- ~~다른 사람이 입력한 정답을 확인 가능함~~

### 🔹 3. 정답 처리

- 가장 먼저 정답을 맞춘 유저가 해당 라운드 점수 N점 획득
- 유효 시간 내 정답자가 없을 경우:
  - 다음 문제로 자동 전환되며 안내 문구 출력
- 총 N개의 문제가 순차적으로 출제됨
- 다른 사람이 입력한 정답의 정답 여부를 확인 가능함

---

### 🔐 제한 및 정책

| 항목           | 내용                                             |
| -------------- | ------------------------------------------------ |
| 정답 제출 횟수 | 무제한 (단, 틀릴 경우 감점)                      |
| 정답 형식      | 문자열 (대소문자 구분 없음, 트림 처리 적용 가능) |
| 부정행위 방지  | 초당 입력 횟수 제한 고려 가능 (optional)         |
